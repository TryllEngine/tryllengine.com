Installation Guide

Get BoscoTryll up and running in minutes

Installation & Launch Guide

  1. 1. First Launch: Run launcher.exe (this will install all dependencies)
  2. 2. Allow Windows Defender to run the script when prompted
  3. 3. Wait for automatic installation of:
    • • Python 3.12 with dependencies
    • • Ollama with model qwen2.5:7b
    • • Mod files into Steam\steamapps\common\Deep Rock Galactic\FSD\Mods\BoscoTryll\
  4. 4. Launch the game and enable BoscoTryll in the mod list
  5. 5. Enable subtitles in Game Settings
  6. 6. Start a mission with Bosco (single-player or multiplayer without players) - Bosco will react to all in-game shouts according to his personality (customizable)

For Subsequent Launches:

Run launcher.exe before starting the game and keep it running in the background while playing for the mod to work.

✓ That's it! BoscoTryll is now installed and ready to use.

Customization Guide

Make Bosco truly yours

Customize Bosco's Personality

Edit Bosco's personality by modifying the configuration file:

Steam\steamapps\common\Deep Rock Galactic\FSD\Mods\BoscoTryll\library\bosco_config.json:

{
  "personality": "Sad and depressed personality. Very lazy and lamenting everything. Clearly not enthusiastic to be here. Every response carries some regret and complaining but reluctingly submissive.",
  "voice": "am_michael"
}

Example Personalities:

Veteran Miner

"Grizzled veteran who's seen it all. Shares war stories and mining wisdom."

Annoyed Old-Timer

"Been doing this job too long. Complains about everything but gets the job done."

Lazy Little Bot

"Would rather be in sleep mode. Does the bare minimum with lots of sighs."

Voice Customization:

Change Bosco's voice by replacing "voice": "am_michael" with one of the available English voices.

System Requirements

Make sure your system meets these specifications

Minimum Requirements

  • GPU

    12GB+ VRAM (Works best on NVIDIA)

  • RAM

    16GB+ system memory

  • Storage

    8GB free space (LLM takes ~6GB)

  • OS

    Windows

  • Game Settings

    Language: English, Subtitles: Enabled

Performance Note

Important Information

LLM is loaded into VRAM at all times and takes up about 9GBs of VRAM. GPU processing resources only used during active text generation.

Performance Tip: If you're experiencing FPS drops, tweak your graphics settings to make room for the LLM.
