**Tryll Assistant v0.0.10 is out** — bringing a smoother first-run experience, smarter dialogue handling, and support for **9 new games**.

![Tryll Assistant v0.0.10 release screenshot](/assets/blog/tryll-assistant-v0-0-10-inline-1.jpg)

## What's New

- **5-step setup wizard** — a guided first-launch flow to get you running fast
- **Dialogue memory** — the assistant now tracks context across follow-up questions for smarter answers
- **Real-time download progress** — model downloads show progress directly on the home screen
- **Updated backend selection** — clearer cards and tags make it easier to choose the right compute backend
- **Dedicated send button** — improved chat input experience
- **Redesigned pages** — Contacts, FAQ, and About pages rebuilt for clarity
- **Better stability** — improved logging and bug fixes across the board
